# AMD RCCL-install and RCCL Test

please use root to install rccl

1- Git clone or download the repository:

  ```
  git clone https://github.com/cpoc-internal/rccl-install
  ```

2. Run the shell script to install the rccl test without accessing the folder, Run this as ROOT

  ```
   chmod +x /root/rccl-install/rccl-install.sh
  sudo -u root /root/rccl-install/rccl-install.sh
 ```

3. Run the shell script to install the rccl test without accessing the folder, Run this as ROOT this is for AMD MI210

  ```
   chmod +x /root/rccl-install/rccl-c845-mi210.sh
  sudo -u root /root/rccl-install/rccl-c845-mi210.sh
 ```
